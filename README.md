
  # Patient Vitals Alert System

  This project is for the CEN3001 - Advanced Computer Programming course at Halic University.
  The system loads data from a synthetic ICU patients vitals signs dataset (source: IEEE DataPort) and alerts medical personnel of critical signs. 

  Features:
  Real-time monitoring of patient vitals

Alerts for abnormal heart rate, blood pressure, temperature, etc.

FastAPI backend for processing and serving data

React frontend with live visualizations

Dataset integration for testing and simulation

Tech Stack

Frontend: React, JavaScript, Vite
Backend: FastAPI (Python)
Database / Dataset: XLSX or CSV
Other: HTML, CSS, Axios, Node.js

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run pip install -r requirements.txt
  uvicorn main:app --reload
